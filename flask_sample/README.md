# Flask Sample (student)

A minimal Flask web app for students to explore routes, templates, forms, and a tiny JSON API.

Quick start

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
# or
# export FLASK_APP=app.py
# flask run
```

Open http://127.0.0.1:5000 in your browser.
